# 物理原型制作

无论你如何着手交互设计项目，再多的数字原型也无法取代实体原型。通常，在设计数字或软件体验之前，项目需要先了解实体体验。例如，考虑一下智能手机的设计。虽然图形用户界面（GUI）通常被认为是交互和设计的主要功能，但它只是一个次要功能，只有在智能手机的物理设计最终确定后才会显现出来。智能手机的形状、外观和触感，都成为设计阶段的重要考量因素。

即使是设计基于软件的输出，例如移动应用，你也常常会先用纸或其他更简单的材料制作实体原型，然后再投入到软件开发中。这是因为实体原型，尤其是用纸制作的，成本低廉、可靠且易于制作/操作。养成习惯，在进入设计流程时，首先制作实体原型。本节将介绍交互设计师可以探索实体原型的不同方法。
