## 任务

观看以上视频，学习如何创建设计原型。完成此任务，您可以选择：

- 重用您之前正在制作的原型/设计（此任务至少需要 3 个屏幕），或者
- 通过复制 Coles、Woolworths、Spotify 或 Ikea 应用中的 3 个屏幕来创建一个新的原型。

尽可能多地重现界面中的交互，包括点击/轻触事件（例如悬停、点击、拖动、滚动）、手势以及内部交互（例如打开下拉菜单或拉出式菜单）。
