# 软件原型设计

当设计一个主要以图形用户界面 (GUI) 形式呈现的交互时，通常会使用 Figma 或 Sketch 等定制原型设计软件。这通常是在经过一系列纸质原型设计阶段，完善细节之后，再投入时间在软件中制作数字原型。不过，在某些情况下，如果界面并不复杂，直接使用软件可能更省时。

## 获取原型设计软件

_请注意，选择哪款软件进行原型设计取决于行业预期和您的设计工作流程。请始终关注最新的设计趋势和技术发展。_

作为本课程的一部分，您将获得完整的 Adob​​e CC 套件的使用权限。在本课程中，如有需要，可以使用 Adob​​e XD 或 Figma 等原型设计软件进行数字原型设计。Figma 在行业中应用更为广泛，并且是一款功能更全面的 UI/UX 数字原型设计工具。 Adobe XD 在某些领域仍有应用，但其发展受限，因此许多行业从业者更倾向于使用 Figma 作为首选工具。

**注意：** 由于新南威尔士大学 (UNSW) 未获得 Figma 的授权，因此 UNSW IT 可能无法提供 Figma 支持。

**注意：** 本课程的其他教程文件可在“软件与学习”部分找到。

## 安装 Figma

Figma 可以完全在浏览器中使用，但也提供桌面版本。您可以选择付费使用 Figma，或者作为学生，您可能有资格享受免费的教育计划。教育计划详情请见此处：[https://help.figma.com/hc/en-us/articles/360041061214-Verify-education-status](https://help.figma.com/hc/en-us/articles/360041061214-Verify-education-status)

- 下载适用于您操作系统的 Figma：[https://www.figma.com/downloads/](https://www.figma.com/downloads/)
- 您需要创建一个 Figma 帐户
- Figma 为学生提供教育帐户，您需要向 Figma 验证您的学生身份，并且可能需要每年重复验证。建议您使用此帐户而非标准帐户，因为您可以免费获得完整的 Figma 功能。创建帐户时请使用您的大学邮箱，以便流程更加顺畅，并准备好相关证明文件（例如成绩单、入学证明或学生证）。 - 点击此链接创建 Figma 学生帐户：[https://www.figma.com/education/higher-education/](https://www.figma.com/education/higher-education/)
- 启动安装程序，并按照屏幕提示安装 Figma
- 您可能需要在稍后阶段安装 Figma 的配套移动应用程序，以便测试您的设计
