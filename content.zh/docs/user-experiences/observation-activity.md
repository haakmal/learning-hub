# 观察活动

这些活动的目的是利用观察所得的洞察来进一步完善你的原型。

- **阅读材料：**《交互设计》- Preece，第 288-297 页，8.6.1 现场直接观察，8.6.2 受控环境下的直接观察

## 任务

- 以你自己为用户，对你最新的原型进行观察测试。
- 在使用原型时，边操作边大声思考，并记录你的感受和行为。
- 反思你的体验，并记下任何需要对你的想法进行的修改。
- 向一位朋友简要介绍“大声思考”的观察技巧。
- 使用“大声思考”的技巧观察你的朋友使用你的原型。
- 记录他们的观察结果，并记下任何额外的修改。
