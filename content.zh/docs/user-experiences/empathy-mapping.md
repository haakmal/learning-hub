# 同理心地图

**请向导师咨询此任务的更多指导。此任务包含两个部分，您可以选择完成其中一项或两项。** 请一位朋友或家人参与此活动。您需要观察他们与您一起完成此活动的过程。他们需要使用电脑或手机，并作为您当天的用户。这是一项同理心地图练习，在您的用户进行活动的过程中，请记录他们所说、所做、所感和所想。您还可以思考他们在体验过程中感到痛苦的地方以及他们希望从中获得什么。更多信息请参阅附件阅读材料。

## 任务2a：评估中的同理心地图绘制

- 完成评估中一位用户的同理心地图绘制要求。首先，您需要为该用户拟定一个基本画像。

- 完成后，分析地图，并设身处地地站在用户的角度思考，以感同身受。您能否找出他们体验中的缺失环节？您会如何改进他们的体验？将这些想法可视化为思维导图可能会有所帮助。

## 任务2b：与用户共情

- **您应该根据评估场景调整此任务**

- 首先，快速拟定用户画像，他们是谁？他们的独特之处是什么？

- 让您的用户打开一个外卖应用（例如 Uber Eats），并体验订餐流程。他们不必完成订单，但如果愿意也可以完成（这将为您提供更多见解）。在绘制同理心地图的过程中，详细记录用户所说、所做、所感和所想。

- 完成后，分析地图，设身处地地站在用户的角度思考问题。您能否找出他们体验中的缺失环节？您会如何改善他们的在线订餐体验？
