# 可供性、可见性和限制

**注意**：关于可供性（affordances）的定义存在诸多混淆，Preece 等人的描述似乎犯了一个常见的错误。可供性首先是一种与事物交互的可能性，无论你是否能够感知到它。他们将其描述为“线索”，这是误导性的。严格来说，只有当可供性（与事物交互的方式）可见（被感知）时，它才能被称为线索。Gaver 的论文（链接如下）对此进行了澄清。

**阅读材料：**

- 《交互设计》- Preece，第 26-30 页，1.7.3 节。设计原则
- 技术可供性 – Gaver ([https://www.lri.fr/~mbl/Stanford/CS477/papers/Gaver-CHI1991.pdf](https://www.lri.fr/~mbl/Stanford/CS477/papers/Gaver-CHI1991.pdf))
