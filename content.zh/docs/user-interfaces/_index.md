## 关于本书

欢迎阅读《用户界面工作手册》！本手册汇集了作为一名交互设计师可以学习的补充资源和重要技能，通过一系列以 UI 设计技能和交互设计流程为核心的可实践任务与活动来展开。本书中的任务最初是作为本课程学生的课后作业，以单独练习单的形式提供。在收集了学生和导师的反馈后，这些任务现已向所有学生开放，作为辅助学习活动，方便你打印出来，在未来的学习过程中或课程期间随时使用。希望在制作这些活动中投入的努力，能够在你学习交互设计的旅程中对你有所帮助！

用户界面设计（User Interface Design，简称 UI）是交互设计的第二个组成部分。你很可能已经接触过 UI/UX 或 用户界面 / 用户体验 这样的术语，这些通常是用来统称交互设计师所从事工作的概括性说法。请记住：没有 UX，UI 是不完整的。

UI 设计是交互设计中更有趣、更具活力的一面！你需要具备视觉传达方面的技能，例如平面设计、出版设计、动画设计以及产品造型设计，才能充分发挥这一技能要求。本手册中的任务将涵盖从版式最佳实践到 UI 设计原则等多个主题。其中许多内容也会在其他视觉设计课程（如平面设计）中涉及，因此本部分的所有任务都默认你已具备基本的视觉传达知识。

## 如何阅读本书

本书中的每一页都是一个任务，旨在帮助你学习交互设计师所需的核心技能。我们为不同的核心技能提供了多本工作手册，包括：用户体验设计（UX）、用户界面设计（UI）、可用性测试以及原型设计。

如果你想尝试某个任务，或该任务已由导师布置，请使用菜单导航到相应的章节编号（例如 *桌面设计*），并仔细阅读任务说明。部分任务可能包含额外阅读材料、需要下载的软件或需要回答的问题，相关链接将在可能的情况下提供。在完成本工作手册中的任务时，请记得在你的过程日志中进行记录，因为你参与的所有活动都将计入课程参与度。
