# 线框图绘制

本次活动的目的是探索线框图绘制和用户流程图绘制。您可以在评估 2 的界面设计中运用此方法。

- **阅读材料：** Preece，《交互设计》，第 491-493 页，13.5 交互设计工具
- **补充阅读材料：** Cao，《什么是线框图？设计你的用户体验主干》，[https://www.uxpin.com/studio/ui-design/what-is-a-wireframe-designing-your-ux-backbone/](https://www.uxpin.com/studio/ui-design/what-is-a-wireframe-designing-your-ux-backbone/)

## 任务

- 选择一个你每天都使用的应用，并绘制其三层线框图。用户有多少种不同的路径？观察最长的路径，是否存在任何捷径？
- 绘制第二个线框图，修改原有路径，使其更加高效（尽量缩短屏幕间的距离，以便用户快速移动）。
- 您可能需要重新设计交互方式，例如滑动、点击等，以使线框图更加高效。请考虑如何在线框图中展示这些交互。
