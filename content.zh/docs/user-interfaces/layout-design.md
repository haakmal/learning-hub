# 布局设计

本课我们将学习布局设计的基础知识。用户界面设计借鉴了视觉设计领域的知识，本任务旨在探索（并挑战）传统的设计方法。

- **阅读材料：**《设计界面》（Designing interfaces），作者：Tidwell，第 209-220 页。“布局基础”部分。可在 Moodle 第五周资源中找到。

## 任务

- 阅读以上章节。重点关注“布局基础”部分，但整章内容都非常有用，值得一读。注意视觉层级和格式塔原则在用户界面设计中的重要性。

- 设计一个社交媒体应用的单屏界面，并展示所有四个格式塔原则。

- 在你的设计中标记出需要体现这四个原则的区域，并简要解释它们是如何在你的设计中体现的。
